How to configure multiple agents / channels

  1. Invite the bot to each Slack channel (/invite @your-bot).
  2. Copy channel IDs (Slack → channel details → Channel ID, e.g. C0…).
  3. Copy agent IDs from WxO (or use answer-only agents for gateway_thread).
  4. Add one entry under bindings: in the Config tab (see multi-channel example).
  5. Save — poller reloads on the next cycle. Prefer reply_mode: gateway_thread + an answer-only agent.
